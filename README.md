# Paper-Install
Paper-Install - простая утилита для установки и быстрого запуска Minecraft-серверов с ядром Paper для множеста версий. Полезен для тех, кто тестирует плагины на поддержку нескольких версий одновременно

### Установка
Для корректной работы у вас должен быть установлен `curl`, а также JDK версии 8, 17 и 21.
```bash
curl -O paper-install https://raw.githubusercontent.com/dertfin0/paper-install/refs/heads/master/paper-intall
chmod +x paper-install
./paper-install
```

При первом запуске вас попросят указать путь к указанным выше версиям JDK, а также путь для установки самих версий сервера. Путь к JDK указывается в виде основной директории, где он расположен без `/` в конце. Путь обязательно должен быть полным, без использования `~`!  
Пример: `/home/your_user/.jdks/corretto-21.0.7`

### Использование
Базовый синтаксис: `./paper-install <версия_jdk> <версия_игры>`

В качестве версии JDK можно указать `8`, `17` или `21`, а версия игры указывается в полном формате, например, `1.16.5`.  
Если указанная версия не была установлена ранее, то она будет автоматически установлена и запущена (с авто-подписью EULA). Если версия была установлена ранее, то она будет автоматически запущена.  

Пример использования:  
Вам нужно установить сервер на версии `1.16.5` и еще один на `1.21.4`.  
Вы запускаете по-очереди:
```bash
./paper-install 8 1.16.5 # т.к. 1.16.5 использует Java 8
./paper-install 21 1.21.4 # т.к. 1.21.4 использует Java 21
```
При первом запуске каждый сервер будет скачан и запущен. Вы можете внести все нужные изменения, а после снова прописать любую из этих команд, чтобы быстро запустить уже установленный и настроенный сервер.

Зачем это нужно? При ручной установке Вам бы пришлось вручную устанавливать jar-файл для каждой версии, создавать папки для серверов, подписывать eula, а также указывать путь к нужной Java при каждом запуске. Здесь вы один раз указываете пути к JDK, а потом просто устанавливаете и запускаете сервер одной командой
